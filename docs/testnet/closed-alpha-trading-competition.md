Pacifica is excited to announce our Weekly Trading competitions! Competitions will begin each Monday at 00:00 UTC and run through the following Friday at 23:59 UTC.

At the conclusion of each weekly competition period, we will announce the winners, as well as any changes to the rule structure for the following week.

Competition Period:

#### 

[](#scoring-metrics)

Scoring Metrics:

*   The Week 1 competition will reward users purely on their absolute USDP PnL.
    
*   PnL = ending\_account\_equity - beginning\_account\_equity - deposits\_during\_period + withdrawals\_during\_period.
    

Please note that this is calculated as USDP PnL, not % ROI. **Therefore it is strongly encouraged that each user starts with a full 10,000 USDP balance.** If you currently have <10,000 USDP, please mint additional test funds, or ask the team to send some to you.
