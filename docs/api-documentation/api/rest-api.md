All example API calls use the Pacifica Mainnet URL, but you can make the same requests against Testnet using the the Pacifica Testnet url.

Mainnet REST endpoint URL: [https://api.pacifica.fi/api/v1](https://api.pacifica.fi/api/v1) Testnet REST endpoint URL: [https://test-api.pacifica.fi/api/v1](https://test-api.pacifica.fi/api/v1)

Requests and responses use JSON.
